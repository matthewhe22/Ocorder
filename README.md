# TOCS Owner Order Portal

A production-ready web portal for purchasing Owner Corporation certificates, built with React and served by a lightweight Node.js HTTP server.

---

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Build the app
npm run build

# 3. Start the server
npm start
```

App will be available at **http://localhost:3000**

## Deployment Options

### Option 1 — Node.js Direct (VPS / Cloud VM)

```bash
npm install
npm run build
NODE_ENV=production PORT=3000 npm start
```

Use **PM2** to keep it running:
```bash
npm install -g pm2
pm2 start server.js --name tocs-portal
pm2 save
pm2 startup
```

---

### Option 2 — Docker

```bash
docker build -t tocs-portal .
docker run -d -p 3000:3000 --name tocs-portal tocs-portal
```

---

### Option 3 — Nginx Reverse Proxy

Run Node on port 3000, then proxy via Nginx:

```nginx
server {
    listen 80;
    server_name portal.tocs.co;

    location / {
        proxy_pass         http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header   Upgrade $http_upgrade;
        proxy_set_header   Connection 'upgrade';
        proxy_set_header   Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

---

### Option 4 — Static Hosting (Netlify / Vercel / S3)

The `dist/` folder is a standard SPA — deploy it to any static host.

**Netlify:** Add a `_redirects` file to `dist/`:
```
/*  /index.html  200
```

**Vercel:** Add `vercel.json`:
```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}
```
